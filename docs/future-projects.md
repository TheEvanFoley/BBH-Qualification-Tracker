# Future Project Workspace

The current qualification tracker is stable and shareable. The next major workstreams should stay separate so they can evolve without destabilizing the live companion app.

## Current Project

- `BBH-Qualification-Tracker`
  Live qualification, benchmark, and trek-priority companion app.

## Upcoming Projects

- `BBH-Site-Analysis`
  Manual-assisted video analysis for sites, variants, mirrors, and anticipatory shot knowledge.

- `BBH-Gameplay-Automation`
  Research repo for parity testing, automation experiments, and TAS-like score analysis.

## Suggested GitHub Repo Names

- `BBH-Qualification-Tracker`
- `BBH-Site-Analysis`
- `BBH-Gameplay-Automation`

## Why Keep Them Separate

- The tracker stays usable and easy to share.
- The site-analysis tool can focus on structured review workflows.
- The automation repo can stay experimental without adding risk to the user-facing tools.
